Papers you can steer

Nine interactive papers, one presenter. Each is a figure you can turn in your hands and a text that walks you through it — press P (or tap) to start the journey, click a point to ask what it is. They run here exactly as they run natively: the same domain code, the same plugins, rasterized in software.
